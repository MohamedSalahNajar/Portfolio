About ARCADIA

A portfolio project demonstrating vanilla JavaScript game development — no frameworks, no dependencies. Each game is built with plain HTML, CSS, and JS, sharing a single stylesheet for a cohesive design. Scores are saved locally in your browser via localStorage.
